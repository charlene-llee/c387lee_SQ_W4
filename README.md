# Week 4 Example 2 | Getting Over It: Hands Edition

## Setup and Interaction Instructions

To run the sketch locally, open `index.html` in Google Chrome using Live Server. Choose between the two buttons presented and try to climb all the way to the top.

**Opening the Chrome Console**

- **Windows:** Press `F12` or `Ctrl + Shift + J`, then click the **Console** tab
- **Mac:** Press `Cmd + Option + J`

The console will show any errors in your sketch.
